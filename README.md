# Finance Data Pipeline - Brasil

## Visão Geral
Este projeto tem como objetivo construir um pipeline de dados financeiros utilizando dados públicos brasileiros, aplicando boas práticas de engenharia de dados, desde a ingestão até a transformação e carga dos dados.

O projeto é parte de um estudo prático voltado ao desenvolvimento de competências em Engenharia de Dados, com foco em organização, escalabilidade e qualidade dos dados.

---

## Fonte de Dados
Os dados utilizados neste projeto são provenientes de bases públicas brasileiras, com foco na área financeira e econômica (ex.: Banco Central do Brasil, Tesouro Nacional ou outras fontes oficiais).

A definição exata das fontes será detalhada conforme a evolução do projeto.
